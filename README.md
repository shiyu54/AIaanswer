一个智能的浏览器答题插件，赛博朋克风格UI，自动识别网页中的题目（单选、多选、判断、填空），通过AI大模型获取答案并自动填写。

## 项目地址

- GitHub: [https://github.com/shiyu54/AIaanswer](https://github.com/shiyu54/AIaanswer)

## 功能特点

- **🤖 AI自动答题**: 使用AI大模型获取问题答案，自动勾选/填写正确答案
- **🧠 联想记忆**: 答过的题自动入库，再次遇到直接复用，节省AI调用
- **📚 智能题库**: 按时间倒序查看所有答题记录，支持搜索、删除、清空
- **📋 答题记录**: 每次作答记入历史，显示题型、答案、来源（AI/题库）、时间
- **🔍 智能题目识别**: 自动识别单选题、多选题、判断题、填空题、下拉题
- **⚡ 题型模板系统**: 预设站点DOM结构，秒级完成题目识别，无需AI分析
- **🎨 赛博朋克主题**: 深灰色背景 + 霓虹青/紫渐变 + 发光动效

## 安装方法

### 开发者模式安装

1. 打开Chrome浏览器，进入 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `ai-answer-assistant` 文件夹
5. 插件安装完成

## 使用方法

1. 打开包含题目的网页
2. 点击插件图标打开控制面板
3. 点击「扫描题目」查看识别到的题目数量
4. 点击「开始自动答题」开始自动答题
5. 答题后可在 📚 题库 查看已收集的题目，在 📋 记录 查看详细日志

## 支持题型

- **单选题**: 带有radio按钮的选择题
- **多选题**: 带有checkbox的选择题
- **判断题**: 正确/错误二选一
- **填空题**: 带有文本输入框的题目
- **下拉题**: 带有select下拉框的题目

## 自定义AI模型

## 常见问题

### Q: 为什么扫描不到题目？

A: 可能是网站的题目结构特殊，插件无法识别。可以尝试刷新页面后重试，或者提供网站URL以添加新的站点模板。

### Q: 答案不正确怎么办？

A: AI答案基于大模型理解，可能存在错误。建议：
- 使用更强大的自定义模型
- 多答几次，重复题目会自动复用已确认的答案
- 手动检查重要题目的答案

### Q: 题库存在哪里？

A: 题库和答题记录都存储在 `chrome.storage.local` 中，跨页面、跨会话持久保留。上限约2000道题，超出自动淘汰最旧条目。

### Q: 无法连接API服务器？

A: 如果出现"无法连接到API服务器"错误，请检查：
1. 网络连接是否正常
2. 代理/VPN设置是否正确
3. API地址和Key是否配置正确

## 注意事项

1. **网站兼容性**: 不同网站的题目结构可能不同，部分网站可能无法正确识别
2. **使用责任**: 请在遵守相关规定的前提下使用本插件
